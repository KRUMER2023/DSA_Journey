838. Push Dominoes ^cPGW2v3j

There are n dominoes in a line, and we place each domino vertically upright. In the beginning, we simultaneously push some of the dominoes either to the left or to the right. ^uHs7OuJu

After each second, each domino that is falling to the left pushes the adjacent domino on the left. Similarly, the dominoes falling to the right push their adjacent dominoes standing on the right. ^Ws2fjcsr

When a vertical domino has dominoes falling on it from both sides, it stays still due to the balance of the forces. ^daJW7aS5

For the purposes of this question, we will consider that a falling domino expends no additional force to a falling or already fallen domino. ^GTuycuoD

You are given a string dominoes representing the initial state where: ^WXqMIalt

dominoes[i] = 'L', if the ith domino has been pushed to the left, ^PYI3islQ

dominoes[i] = 'R', if the ith domino has been pushed to the right, and ^4wrEbNbn

dominoes[i] = '.', if the ith domino has not been pushed. ^e0TpDudj

Return a string representing the final state. ^j7lfa6lZ

Example 1: ^59RRxZsm

Input: dominoes = "RR.L" ^oYUvbWCn

Output: "RR.L" ^iLAHeoBP

Explanation: The first domino expends no additional force on the second domino. ^RDFTb2sh

Example 2: ^O2Xuv9bZ

Input: dominoes = ".L.R...LR..L.." ^Z8L7Uv4q

Output: "LL.RR.LLRRLL.." ^HVQlHxKf

.L.R...LR..L.. ^dIBcyVzA
